# 箱子类型

> 版本: 1.0.0  
> 源码: `Chips-SDK/src/types/box.ts`

## 主要类型

### BoxMetadata

- `chip_standards_version: ProtocolVersion`
- `box_id: ChipsId`
- `name: string`
- `created_at: Timestamp`
- `modified_at: Timestamp`
- `layout: string`
- `tags?: Tag[]`
- `description?: string`
- `author?: string`
- 允许扩展字段

### BoxCardInfo

- `id: ChipsId`
- `location: LocationType`
- `path: string`
- `filename: string`

### BoxStructure

- `cards: BoxCardInfo[]`

### BoxContent

- `active_layout: string`
- `layout_configs: Record<string, unknown>`

### Box

- `id: ChipsId`
- `metadata: BoxMetadata`
- `structure: BoxStructure`
- `content: BoxContent`

### CreateBoxOptions / UpdateBoxOptions

用于创建与更新:

- `name`, `layout`, `tags`, `description`
- `metadata?: Partial<BoxMetadata>`

### QueryBoxOptions

- `tags?`, `layout?`, `name?`
- `createdAfter?`, `createdBefore?`
- `limit?`, `offset?`

### AddCardToBoxOptions / ReorderCardsOptions

- `AddCardToBoxOptions`: { cardId, location, path?, position? }
- `ReorderCardsOptions`: { cardIds: ChipsId[] }

### LayoutConfig

用于布局配置:

- `type: string`
- `columns?`, `gap?`, `padding?`, `itemWidth?`, `itemHeight?`
- 允许扩展字段
