# 资源类型

> 版本: 1.0.0  
> 源码: `Chips-SDK/src/resource/types.ts`

## 资源类型

- `ResourceType`: `'image' | 'video' | 'audio' | 'font' | 'document' | 'other'`
- `ResourceState`: `'pending' | 'loading' | 'loaded' | 'error'`

## ResourceInfo

- `id: string`
- `name: string`
- `path: string`
- `type: ResourceType`
- `mimeType: string`
- `size: number`
- `checksum?: string`
- `created?: string`
- `modified?: string`

## CacheEntry

- `info: ResourceInfo`
- `data: Blob | ArrayBuffer`
- `objectUrl?: string`
- `state: ResourceState`
- `lastAccess: number`
- `refCount: number`

## ResourceLoadOptions

- `cache?: boolean`
- `priority?: 'high' | 'normal' | 'low'`
- `timeout?: number`
- `onProgress?: (loaded, total) => void`

## ResourceManagerOptions

- `maxCacheSize?: number`
- `maxCacheCount?: number`
- `cacheTTL?: number`
- `autoCleanup?: boolean`

## ResourceUploadOptions

- `path: string`
- `overwrite?: boolean`
- `onProgress?: (loaded, total) => void`

## PreloadOptions

- `priority?: 'high' | 'normal' | 'low'`
- `concurrency?: number`
