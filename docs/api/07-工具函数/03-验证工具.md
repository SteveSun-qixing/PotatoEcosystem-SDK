# 验证工具

> 版本: 1.0.0  
> 源码: `Chips-SDK/src/utils/validation.ts`

## 方法

- `validateCardMetadata(metadata): metadata is CardMetadata`
- `validateBoxMetadata(metadata): metadata is BoxMetadata`
- `validateProtocolVersion(version: string): boolean`
- `validateTimestamp(timestamp: string): boolean`
- `validateTag(tag: unknown): boolean`
- `validateTags(tags: unknown): boolean`
- `validateResourcePath(path: string): boolean`
- `validateFileSize(size: number, maxSize: number): boolean`
- `validateMimeType(mimeType: string): boolean`
- `validateUrl(url: string): boolean`
- `validateEmail(email: string): boolean`

## 重点规则

- `validateResourcePath` 要求路径以 `resources/` 开头且不能包含 `..`.
- `validateTimestamp` 通过 `new Date(timestamp)` 判断, 仅保证可解析.
