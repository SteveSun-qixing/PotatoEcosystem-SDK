# 卡片类型

> 版本: 1.0.0  
> 源码: `Chips-SDK/src/types/card.ts`

## 主要类型

### CardMetadata

包含卡片基本信息:

- `chip_standards_version: ProtocolVersion`
- `card_id: ChipsId`
- `name: string`
- `created_at: Timestamp`
- `modified_at: Timestamp`
- `theme?: string`
- `tags?: Tag[]`
- `description?: string`
- `author?: string`
- 允许扩展字段

### CardManifest

- `card_count: number`
- `resource_count: number`
- `resources: ResourceInfo[]`

### CardStructure

- `structure: BaseCardInfo[]`
- `manifest: CardManifest`

### Card

完整卡片对象:

- `id: ChipsId`
- `metadata: CardMetadata`
- `structure: CardStructure`
- `resources: Map<string, Blob | ArrayBuffer>`

### CreateCardOptions / UpdateCardOptions

用于创建/更新:

- `name`, `type`, `theme`, `tags`, `description`
- `metadata?: Partial<CardMetadata>`

### QueryCardOptions

查询条件:

- `tags?`, `type?`, `name?`
- `createdAfter?`, `createdBefore?`
- `modifiedAfter?`, `modifiedBefore?`
- `limit?`, `offset?`

### BaseCardData / CardResource

- `BaseCardData`: 用于基础卡片结构化数据
- `CardResource`: 用于资源描述与数据
