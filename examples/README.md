### 文件操作示例
- [file-operations/](./file-operations/) - 文件操作示例

### 插件示例

## 运行示例

### Web示例

```bash
cd examples/basic
npx vite
```

### Node.js示例

```bash
cd examples/file-operations
npm install
node index.js
```
