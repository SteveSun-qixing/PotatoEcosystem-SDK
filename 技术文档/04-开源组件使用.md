# 开源组件使用

Chips SDK 使用多个成熟的开源组件来实现核心功能。

## 1. 核心依赖

### 1.1 @chips/core
- **用途**：薯片内核
- **许可证**：MIT
- **版本**：^1.0.0

### 1.2 eventemitter3
- **用途**：事件系统
- **许可证**：MIT
- **版本**：^5.0.0

## 2. 渲染相关

### 2.1 Video.js
- **用途**：视频播放
- **许可证**：Apache 2.0
- **版本**：^8.0.0

### 2.2 Prism.js
- **用途**：代码高亮
- **许可证**：MIT
- **版本**：^1.29.0

### 2.3 KaTeX
- **用途**：数学公式渲染
- **许可证**：MIT
- **版本**：^0.16.0

## 3. 编辑器相关

### 3.1 Quill / ProseMirror / TipTap
- **用途**：富文本编辑
- **许可证**：MIT
- **版本**：根据选择

## 4. 工具库

### 4.1 lodash-es
- **用途**：工具函数
- **许可证**：MIT
- **版本**：^4.17.21

### 4.2 date-fns
- **用途**：日期处理
- **许可证**：MIT
- **版本**：^2.30.0

所有开源组件的许可证信息会在 SDK 中声明，遵守各组件的许可协议。
