# 基础类型

> 版本: 1.0.0  
> 源码: `Chips-SDK/src/types/base.ts`

## 类型列表

### ChipsId

- `string`
- 格式: `[0-9a-zA-Z]{10}`

### Timestamp

- `string`
- ISO 8601 时间戳, 例如 `2026-02-01T12:00:00.000Z`

### Tag

- `string` 或 `tuple`
- 允许 `['key', 'value1', 'value2']` 形式

### ProtocolVersion

- 模板字符串: `${number}.${number}.${number}`

### Status

- `'success' | 'error' | 'partial'`

### LogLevel

- `'debug' | 'info' | 'warn' | 'error'`

### FileType

- `'card' | 'box' | 'resource' | 'unknown'`

### LocationType

- `'internal' | 'external'`

### SortDirection / SortOptions

- `SortDirection`: `'asc' | 'desc'`
- `SortOptions`: `{ field: string; direction: SortDirection }`

### PaginationOptions / PaginatedResult

- `PaginationOptions`: `{ limit: number; offset: number }`
- `PaginatedResult<T>`: `{ items; total; limit; offset; hasMore }`
