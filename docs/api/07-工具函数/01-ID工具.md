# ID 工具

> 版本: 1.0.0  
> 源码: `Chips-SDK/src/utils/id.ts`

## 方法

### `generateId(): string`

生成 10 位 62 进制随机 ID.

### `isValidId(id: string): boolean`

验证 ID 是否符合 `[0-9a-zA-Z]{10}`.

### `generateUuid(): string`

生成 UUID v4.

### `isValidUuid(uuid: string): boolean`

验证 UUID v4 格式.

### `generateShortId(): string`

生成 8 位短 ID.
