## 🚀 使用示例

### 基础使用

```typescript
import { ChipsSDK } from '@chips/sdk';

// 创建SDK实例
const chips = new ChipsSDK();

// 加载卡片
const card = await chips.loadCard('path/to/card.card');

// 渲染卡片
await chips.renderCard(card, '#container');

// 设置主题
chips.setTheme('dark');

// 切换语言
chips.setLanguage('en-US');
```

### 高级功能

```typescript
// 使用插件
import myPlugin from './my-plugin';
chips.usePlugin(myPlugin);

// 批量操作
const cards = await chips.loadCards(['c1.card', 'c2.card', 'c3.card']);

// 事件监听
chips.on('card:load', (card) => {
  console.log('卡片已加载:', card.metadata.name);
});

// 自定义配置
chips.setConfig('cache.maxSize', 200 * 1024 * 1024); // 200MB
```

### CLI使用

```bash
# 创建卡片
chips create my-card.card --name "我的卡片"

# 格式转换
chips convert card.card output.md

# 查看信息
chips info card.card --verbose

# 验证格式
chips validate ./cards/ --recursive

# 列出卡片
chips list ./cards/ --sort size
```
