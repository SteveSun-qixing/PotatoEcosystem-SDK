# 产品概述

## 1. 产品定义

Chips SDK（Software Development Kit）是一套完整的开发工具包,让第三方开发者能够轻松地将 Chips 的核心能力集成到自己的应用中。SDK 封装了卡片解析、渲染、编辑等底层逻辑和通用能力,提供简洁易用的 API,使开发者只需极少代码即可实现复杂的卡片功能。

## 2. 产品定位

### 2.1 核心定位

Chips SDK 定位为：
- **开发者工具**：面向第三方开发者的技术产品
- **能力封装**：封装 Chips 生态的核心能力
- **易用优先**：API 设计简单直观,学习成本低
- **跨语言支持**：提供多种编程语言的绑定
- **完整生态**：从开发到部署的完整工具链

### 2.2 与其他产品的关系

```
Chips 生态层次结构

薯片内核 (Core)
    ↓ 调用
Chips SDK
    ↓ 集成
第三方应用
    ↓ 使用
最终用户
```

- **内核层**：薯片内核提供底层能力
- **SDK 层**：封装内核能力,提供开发者 API
- **应用层**：第三方开发者使用 SDK 开发应用
- **用户层**：最终用户使用这些应用

## 5. 产品特色

### 5.1 极简 API 设计

```javascript
// 只需几行代码即可渲染卡片
import { ChipsSDK } from '@chips/sdk';

const chips = new ChipsSDK();
await chips.renderCard('card.card', '#container');
```

### 5.2 完整功能封装

SDK 封装了所有核心功能：
- **文件操作**：读取、写入、解析卡片文件
- **渲染引擎**：完整的卡片渲染能力
- **编辑器**：所见即所得的编辑功能
- **主题系统**：丰富的主题和样式定制
- **插件系统**：扩展和定制功能
- **资源管理**：自动处理文件和资源

### 5.3 多语言绑定支持

提供多种编程语言的SDK绑定：
- **JavaScript/TypeScript**：Web 和 Node.js
- **Python**：桌面应用和数据处理
- **Swift**：iOS 和 macOS 应用
- **Kotlin/Java**：Android 应用
- **C#**：Windows 应用和游戏
- **Go**：服务端应用

### 5.4 系统多语言支持

SDK完全集成薯片生态的系统统一多语言方案：

- **零硬编码**：SDK内部和示例代码都遵循零硬编码原则
- **词汇编码系统**：使用系统词汇编码（i18n.xxx.xxxxxx）
- **自动替换**：开发时使用key，打包时自动替换为编码
- **配置表管理**：所有文本通过配置表管理

> **详细规范**: 参见 [多语言系统规范](../../生态共用/11-多语言系统规范.md)

**支持的语言**：
- 简体中文（zh-CN）
- 繁体中文（zh-TW）
- 英文（en-US）
- 日文（ja-JP）
- 韩文（ko-KR）

**SDK多语言API**：
```javascript
import { ChipsSDK } from '@chips/sdk';

const sdk = new ChipsSDK();

// 设置语言
await sdk.setLanguage('en-US');

// 获取当前语言
const lang = sdk.getLanguage();

// SDK内部所有文本、错误信息、提示都会自动切换
```

### 5.4 跨平台兼容

一套 SDK,多平台运行：
- **Web**：浏览器中直接使用
- **桌面**：Windows、macOS、Linux
- **移动**：iOS、Android
- **服务端**：Node.js、Python、Go 等

### 5.5 完善的工具链

- **CLI 工具**：命令行快速开发
- **开发服务器**：实时预览和调试
- **代码生成器**：自动生成模板代码

## 6. 应用场景

### 6.1 内容管理系统

**场景描述**：
在 CMS 中集成 Chips 卡片作为内容块,用户可以创建、编辑和展示各种类型的富内容。

**SDK 使用**：
```javascript
// 在 CMS 编辑器中嵌入卡片编辑器
const editor = chips.createEditor('#editor', {
  mode: 'inline',
  toolbar: true,
  autoSave: true
});

// 保存卡片内容
const cardData = await editor.getCardData();
await cms.saveContent(cardData);
```

### 6.2 笔记应用

**场景描述**：
笔记应用使用 Chips 卡片作为笔记单元,支持文本、图片、代码、清单等多种内容类型。

**SDK 使用**：
```javascript
// 创建新笔记卡片
const note = await chips.createCard({
  type: 'composite',
  children: [
    { type: 'richtext', content: '会议记录' },
    { type: 'list', items: ['议题1', '议题2'] },
    { type: 'image', url: 'whiteboard.jpg' }
  ]
});

await note.save('meeting-notes.card');
```

### 6.3 博客平台

**场景描述**：
博客使用 Chips 卡片作为文章格式,提供丰富的内容展示和交互体验。

**SDK 使用**：
```javascript
// 在博客中渲染文章卡片
await chips.renderCard(articlePath, '#article-content', {
  theme: 'blog-theme',
  readOnly: true,
  enableComments: true
});
```

### 6.4 在线教育平台

**场景描述**：
课程内容使用卡片组织,支持视频、文档、测验等多种教学材料。

**SDK 使用**：
```javascript
// 创建课程内容
const lesson = await chips.createBox({
  name: 'JavaScript 基础',
  layout: 'list',
  cards: [
    { type: 'video', url: 'lesson1.mp4', title: '变量和类型' },
    { type: 'markdown', content: '## 知识点总结...' },
    { type: 'quiz', questions: [...] }
  ]
});
```

### 6.5 知识库系统

**场景描述**：
企业知识库使用卡片组织文档,支持版本管理、权限控制和协作编辑。

**SDK 使用**：
```javascript
// 加载和展示知识库文档
const doc = await chips.loadCard('docs/api-guide.card');
await chips.renderCard(doc, '#doc-viewer', {
  mode: 'wiki',
  toc: true,
  breadcrumb: true
});
```

### 6.6 移动应用

**场景描述**：
移动 App 使用 SDK 支持查看和编辑卡片内容。

**SDK 使用**：
```swift
// iOS/Swift 示例
import ChipsSDK

let chips = ChipsSDK()
try await chips.renderCard(
    path: "note.card",
    in: containerView,
    options: RenderOptions(theme: .system)
)
```

## 7. 技术特性

### 7.1 模块化设计

SDK 采用模块化设计,开发者可按需引入：

```javascript
// 完整引入
import ChipsSDK from '@chips/sdk';

// 按需引入
import { CardRenderer } from '@chips/sdk/renderer';
import { CardEditor } from '@chips/sdk/editor';
import { FileManager } from '@chips/sdk/file';
```

### 7.2 TypeScript 优先

完整的类型定义和 IntelliSense 支持：

```typescript
interface Card {
  id: string;
  type: CardType;
  content: CardContent;
  metadata: CardMetadata;
}

interface RenderOptions {
  theme?: Theme | string;
  readOnly?: boolean;
  interactive?: boolean;
}
```

### 7.3 响应式设计

自动适配不同设备和屏幕尺寸：

```javascript
// SDK 自动处理响应式布局
await chips.renderCard('card.card', '#container', {
  responsive: true,  // 默认启用
  breakpoints: {
    mobile: 480,
    tablet: 768,
    desktop: 1024
  }
});
```

### 7.4 插件生态

支持扩展 SDK 功能：

```javascript
// 使用插件扩展功能
import LaTeXPlugin from '@chips/plugin-latex';
import MermaidPlugin from '@chips/plugin-mermaid';

chips.use(LaTeXPlugin);
chips.use(MermaidPlugin);
```

### 7.5 事件驱动

丰富的事件系统便于集成：

```javascript
// 监听卡片事件
chips.on('card:load', (card) => {
  console.log('Card loaded:', card.id);
});

chips.on('card:edit', (changes) => {
  console.log('Card edited:', changes);
});

chips.on('error', (error) => {
  console.error('SDK error:', error);
});
```
