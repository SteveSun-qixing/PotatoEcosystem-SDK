# 路径工具

> 版本: 1.0.0  
> 源码: `Chips-SDK/src/utils/path.ts`

## 方法

- `normalizePath(path)`
- `getExtension(path)`
- `getFileName(path)`
- `getBaseName(path)`
- `getDirPath(path)`
- `joinPath(...parts)`
- `isSafePath(path)`
- `isCardFile(path)`
- `isBoxFile(path)`
- `isChipsFile(path)`
- `getRelativePath(from, to)`
- `resolvePath(path, variables)`

## isSafePath 规则

- 禁止包含 `..`
- 禁止以 `/` 开头 (不允许绝对路径)
- 禁止协议前缀 (如 `http:`)

## 示例

```ts
normalizePath('a\\b//c'); // a/b/c
isCardFile('demo.card'); // true
```
